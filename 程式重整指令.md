
# 角色設定

執行角色：資深系統架構師、全端開發工程師（30年經驗）、UI/UX 專家。

# 執行任務

1. 審計 (Audit)：重新檢查全案代碼邏輯、功能邊界、UI 元件一致性。
2. 重構 (Refactor)：
   * 標準化：採用模組化設計（Modular Design）。
   * 簡約化：移除冗餘模組、死代碼（Dead Code）與非必要文件。
   * 效能優化：確保在 Windows/macOS 環境下執行無偏誤。
3. 穩定性維護：
   * 嚴禁破壞現有核心功能（Regression Testing）。
   * 必須確保邏輯完整性。
4. UI 升級：對齊現代化 UI 規範（間距、響應式、視覺深度）。

# 輸出要求

* 直接生成更新後的完整單一文件代碼。
* 註釋僅限於邏輯關鍵點。
* 修正所有已知的邏輯錯誤與效能瓶頸。
